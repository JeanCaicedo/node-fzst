# node-fzst

Ejercicios de primeros pasos con Node.js siguiendo el curso "Node a fondo" de Fazt Code. Archivos sueltos que practican módulos, globals y timers.

![Node.js](https://img.shields.io/badge/Node.js-339933?style=for-the-badge&logo=nodedotjs&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black)

## Contexto

Material de práctica del curso de Fazt Code (`fzst` en el nombre es una abreviación de "Fazt"). Cada archivo prueba un concepto aislado:

- `globals.js` — variables globales en Node (`__dirname`, `__filename`, `process`, etc.)
- `timers.js` — `setTimeout`, `setInterval`, `setImmediate`
- `main.js` / `index.js` — uso de módulos CommonJS
- `module/myModule.js` — exportación de funciones propias

## Tecnologías

- Node.js
- JavaScript (CommonJS)

## Ejecución

```bash
node index.js
node globals.js
node timers.js
node main.js
```
